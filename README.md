# Sculptor
Escultor 3D em C++ | Voxel Engine &amp; Mesh Exporter:  Implementação de um motor de modelagem volumétrica estruturado sobre alocação dinâmica de matrizes tridimensionais. A ferramenta converte coordenadas lógicas de voxels em geometria explícita, exportando modelos otimizados no formato .OFF para renderização em softwares como o MeshLab.
